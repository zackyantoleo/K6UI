Alur Request

Susun request yang dijalankan k6. Gunakan {{nama_variabel}} di URL, header, atau body untuk menyisipkan nilai yang diekstrak dari respons sebelumnya.

Tentukan jumlah virtual user (VU) dan pola beban selama test berlangsung.

Mode Beban

Panduan Durasi

30s30 detik
5m5 menit
1h1 jam
1h30m1 jam 30 menit

Mode Bertahap: setiap stage mendefinisikan transisi dari VU saat ini ke jumlah VU target dalam rentang waktu tertentu.

Test dianggap gagal (exit code ≠ 0) jika target performa ini tidak terpenuhi — berguna untuk CI/CD pipeline.

Target Performa

95% request harus selesai di bawah nilai ini.
Persentase request yang diizinkan gagal.

Apa itu Threshold?

Threshold adalah syarat kelulusan test. Jika tidak terpenuhi, k6 keluar dengan exit code 99.

Contoh: jika kamu set p95 = 500ms dan 5% request melebihi itu, test dinyatakan GAGAL.
