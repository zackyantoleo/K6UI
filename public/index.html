What are Thresholds?

Thresholds are the pass criteria for the test. If they are not met, k6 exits with exit code 99.

Example: if you set p95 = 500ms and 5% of requests exceed it, the test is marked FAILED.

Variables

              

              
            


            

              
How it works

              
Reference a variable as {{name}} in a URL, header, or body — e.g. {{base_url}}/users.

              
Names must start with a letter or underscore and contain only letters, digits, and underscores.

              
If a request extracts a variable with the same name, the extracted value overrides the global one for the requests that follow.

Headers

              

              
            


            

              
How it works

              
These headers are sent with every request in the scenario, including Before and After sub-requests.

              
Values can use {{name}} variables — e.g. Bearer {{token}} with a global or extracted variable.

              
If a request defines a header with the same name, the request's value overrides the global one (names are case-insensitive).
